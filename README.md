# 𝔗𝔥𝔢 ℭ𝔲𝔯𝔰𝔢 𝔬𝔣 𝔱𝔥𝔢 𝔖𝔥𝔞𝔡𝔬𝔴 ℑ𝔰𝔩𝔢

A thrilling text-based adventure game built with **Python**. Face your fate, choose your path wisely, and try to find the legendary Golden Treasure!

## 🎮 How to Play
1. Run the script in any Python environment.
2. Read the story and the challenges carefully.
3. Type your choices (e.g., **red**, **blue**, **gold**) when prompted.
4. Try to survive the traps and reach the treasure!

## ✨ Features
* **Immersive Story:** Mysterious atmosphere with ancient gates and hidden chambers.
* **Stylish Pirate Font:** Beautiful visual title and emojis for a fun experience.
* **User Input Validation:** Handles inputs smoothly using `.lower()` and logic checks.
* **Multiple Endings:** You can either win the treasure or face a deadly trap!

## 🛠️ Technologies Used
* **Python 3.x**
* **Logic:** If/Else statements, Nested Conditions.
